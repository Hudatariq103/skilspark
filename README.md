# SkillSpark Website

## 📌 Project Description
SkillSpark is an educational website designed to help students learn, build, and launch their tech skills. The website provides a clean, modern interface with easy navigation and interactive elements, making it simple for users to explore courses and resources.

## 🚀 Features
- **Responsive Navbar**: Easy navigation across all pages, works on desktop and mobile.
- **Hero Section**: A welcoming banner that highlights the website’s purpose.
- **Course Cards**: Organized course previews with title, description, and “Learn More” buttons.
- **Footer Section**: Includes links and contact information.
- **Mobile Responsive Design**: Optimized layout for different screen sizes.
- **Dropdown Menus and Buttons**: Interactive elements for better usability.
- **Modern Styling**: Clean color scheme and visually appealing layout using CSS and Bootstrap.

## 🛠 Technology Used
- HTML5
- CSS3
- Bootstrap 5
- Git & GitHub (for version control)
